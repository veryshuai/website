Password free ssh login with Ubuntu 12.04

on client:

  1. generate ssh key (ssh-keygen)
    • Everything default, except enter a password when prompted
  2. add identity to ssh-agent (ssh-add)
  3. Copy the key to the remote host (ssh-copy-id)

              ssh-keygen
              ssh-add
              ssh-copy-id

on remote host:

  1. make permissions compliant. ssh doesn't want group to be able to write to any super directory up to home.

              chmod 700 ~/.ssh/* && chmod 600 ~/.ssh/authorized_keys && chmod g-w /home/yourusername
